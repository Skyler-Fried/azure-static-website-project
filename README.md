# azure-static-website-project
My first Azure project using Blob Storage static website hosting.

## Overview
This project demonstrates how to host a static website using Azure Blob Storage static website hosting.

## Services Used
- Azure Storage Account
- Azure Blob Storage
- Static Website Hosting

## Project Tasks
- Created a storage account
- Enabled static website hosting
- Uploaded website files
- Verified public website access
- Updated website content and validated changes

## What I Learned
- How Azure Storage Accounts work
- How static websites can be hosted without a virtual machine
- How to deploy and update website content in Azure
- Basic cloud deployment workflows
